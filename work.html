Spend Sight

iOS Finance App

Receipt scanning and spending insights iOS app.

View Project Website GitHub

Click card to read more

Spend Sight is an iOS app that helps users turn paper receipts into clear spending insights. Users can take a photo of a receipt or upload one from their phone, and the app automatically reads the receipt and extracts important details like the store name, total amount spent, and date. After scanning, users can choose a spending category and save the receipt. The app then tracks daily and weekly spending through simple visual graphs, helping users understand where their money is going and build better financial habits.

Cheap Flight Ticket Finder

Full-Stack Web App

Full-stack flight search app with cheap-first results.

Live Demo GitHub

Click card to read more

Cheap Flight Finder is a full-stack web application that allows users to search for flight options between airports around the world. Users can select an origin airport, destination airport, and departure date to view available flight offers. The application retrieves flight data using the Amadeus Flight Offers API and displays results including the airline, price, departure and arrival airports, and flight times. Results are automatically shown starting from the cheapest flights, helping users quickly compare affordable options. To improve usability, the search fields include an airport autocomplete feature that allows users to search by city, airport name, country, or IATA code. The backend server securely handles API requests and processes the results before sending them to the frontend for display. Note: This project uses the Amadeus Sandbox API, so the flights displayed do not reflect real-time airline availability.
